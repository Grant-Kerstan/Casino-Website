# Casino-Website

A browser-based website built with HTML, CSS, and JavaScript that features interactive Blackjack and Poker games using a shared virtual currency system.

## Features
- Playable Blackjack and Poker games implemented in vanilla JavaScript
- Shared in-game currency system used across both games
- Interactive UI elements and real-time game state updates
- Recovery mechanic that allows users to earn money by interacting with on-screen elements when funds reach zero

## Technologies Used
- HTML
- CSS
- JavaScript
